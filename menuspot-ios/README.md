# MenuSpot — iOS App Build Guide
## Install for FREE on your iPhone (No App Store, No $99)

---

## What You Need
| Requirement | Where to Get |
|---|---|
| Mac computer | Required (Xcode only runs on Mac) |
| Free Apple ID | appleid.apple.com |
| Xcode | Mac App Store (free) |
| Node.js 18+ | nodejs.org (free) |
| iPhone | Your phone |
| AltStore or Sideloadly | altstore.io / sideloadly.io (free) |

---

## STEP 1 — Set Up the Project

Open Terminal and run these commands one by one:

```bash
# 1. Go into the project folder
cd menuspot-ios

# 2. Install all dependencies
npm install

# 3. Install Capacitor CLI globally
npm install -g @capacitor/cli

# 4. Build the web app
npm run build

# 5. Add the iOS platform
npx cap add ios

# 6. Sync your files into the iOS project
npx cap sync ios
```

---

## STEP 2 — Open in Xcode

```bash
npx cap open ios
```

This opens Xcode automatically with your iOS project.

---

## STEP 3 — Configure Xcode (One Time)

Inside Xcode:

1. Click on **"App"** in the left sidebar (the top-level project file)
2. Under **Signing & Capabilities** tab:
   - Check **"Automatically manage signing"**
   - Under **Team**: click the dropdown → **Add Account** → sign in with your **free Apple ID**
   - Set **Bundle Identifier** to something unique like: `com.yourname.menuspot`
3. Connect your iPhone via USB cable
4. Select your iPhone from the device list at the top of Xcode
5. Press **▶ Run** (or Cmd+R)

Xcode will build and install the app on your iPhone!

---

## STEP 4 — Trust the App on iPhone

After installing, iOS will say "Untrusted Developer". Fix it:

1. Go to **Settings → General → VPN & Device Management**
2. Tap your **Apple ID email**
3. Tap **"Trust [your email]"**
4. Open MenuSpot — it works! ✅

---

## STEP 5 — Keep It Working (Re-sign Every 7 Days)

With a free Apple ID, apps expire every **7 days**. Fix this with AltStore:

### Install AltStore (one-time setup):
1. Download **AltStore** from altstore.io on your Mac
2. Install **AltServer** on your Mac (stays running in menu bar)
3. Connect iPhone → Trust Mac
4. In AltServer menu → **Install AltStore** → select your iPhone

### Refresh the app:
- Open **AltStore** on iPhone → tap **MenuSpot** → **Refresh**
- Or connect to the same WiFi as your Mac — AltStore auto-refreshes!

---

## Alternative: Sideloadly (Simpler)

If AltStore feels complex:

1. Download **Sideloadly** from sideloadly.io
2. Build the `.ipa` in Xcode: **Product → Archive → Distribute App → Development → Export**
3. Open Sideloadly → drag the `.ipa` → connect iPhone → click **Start**
4. Trust the app in iPhone Settings (same as Step 4)
5. Repeat every 7 days

## Updating the App

When you make changes to MenuSpot:

```bash
npm run build        # rebuild web files
npx cap sync ios     # sync into iOS project
```

Then press Run in Xcode again.

## Troubleshooting

| Problem | Solution |
|---|---|
| "No devices found" | Unlock iPhone, tap "Trust This Computer" |
| Build fails with signing error | Check Apple ID in Xcode → Signing & Capabilities |
| App crashes on launch | Check Xcode console for error messages |
| Safe area looks wrong | Already patched in ios-patches.css |
| Input text too small | Already patched (font-size: 16px on inputs) |
